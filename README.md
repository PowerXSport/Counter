# Counter
Source code and 3d models for the PowerX rep counter

# GX16 Wiring
| Component      | Pin |
|----------------|-----|
| Click Button   |  1  |
| Plus  Button*  |  2  |
| Minus Button*  |  3  |
| +3.3v **       |  8  |

*The rocker +/- switch activates the opposite pole

**Wire one wire from button, and center pole from +/- to here
